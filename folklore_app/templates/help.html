Структура архива
Архив включает две части - собственно архив и корпус лингвистический.
Навигация
  • Верхняя панель навигации
  • Вкладка "Подробнее" (о дополнительных страницах см. ниже)
  • Войти (для зарегистрированных пользователей)
По каким параметрам можно искать?
Поиск по архиву разделен на две части - простой поиск и поиск расширенный, который содержит больше различных настроек.
Простой поиск
  • Ключевые слова в режиме И/ИЛИ (одновременно все либо хотя бы одно из)
  • Исключение ключевых слов - те слова, которых не должно быть в выдаче
  • Жанр текста (для части текстов)
  • Наличие медиа (по умолчанию - все, при выборе - только с медиа, т.е. фото или видео)
  • Год записи текста
  • Пол, год рождения информанта
  • Место сбора данных. Можно выбрать несколько разных мест. По умолчанию - любое место записи
Расширенный поиск

В дополнение к простому поиску:

  • Опросник, вопрос, подвопрос - для текстов экспедиций, которые работают с опросниками
  • Конкретный информант (по коду)
  • Место рождения информанта (доступно не для всех)
  • Место проживания информанта (может не совпадать с местом записи, например, если кто-то приехал в гости)
Как использовать фильтры поиска?
  • Поля ID, год: номер/год, например, 1970
  • Ключевые слова: начните вводить слово, в выпадающем списке появятся подсказки, из которых можно выбрать нужное.
    Для того, чтобы убрать ненужное слово, нажмите на крестик на прямоугольнике слова.
  • Географическая информация: выберите один или несколько вариантов. Если вы выбираете область, то обновляются имена районов, затем если конкретный район, то предлагаются населенные пункты. Для выбора нажмите на вариант. Он должен выделиться цветом. Для отмены нажмите еще раз, тогда выбор снимется и вариант снова станет белым. Для выбора нескольких вариантов таким же образом выберите те, которые вам интересны.
  • Опросники: варианты выбираются так же, как и в географии. Для выбора нажмите на вариант (или несколько). Для снятия выбора нажмите еще раз.
  • Медиа: для выбора текстов только с медиафайлами (фото или видео) отметьте галочку в этом пункте.
Результаты поиска
  • Таблица результатов поиска.
    В верхней части находится описание параметров поиска, затем указано число наиденных текстов и располагается навигация по страницам выдачи. В таблице текстов идентификатор текста, год, место сбора данных, указание на информанта (код, пол, год рождения). Далее информаци об опросниках и ключевых словах. В последней части начало текста. По ссылке "читать полностью" можно перейти на страницу данного текста.

  • Скачивание результатов выдачи. Для скачивания доступны два формата - простой текстовый и формат JSON (для удобства автоматической обработки).
  • Текст: в левой части - основная информация о тексте (место записи, номер и т.д) и информация об информантах, вопросы, которые перед ними ставились и список ключевых слов. В конце есть обозначение, как можно ссылаться на этот текст (например, ФА ВШЭ 282, то есть Фольклорный архив ВШЭ текст с идентификатором 282).
  • Видео: видео находится под текстом, можно воспроизводить прямо на странице
  • Фотографии: фотографии находятся после видео (или текста). Для того, чтобы увеличить изобрадение, нужно щелкнуть на него и оно откроется на новой странице в полном размере.
Дополнительные страницы
Во вкладке "Дополнительно" есть ссылки на другие страницы.
  • Опросники: страница с опросниками содержит вопросы, которые задавались в рамках экспедиций под руководством А.Б. Мороза. Для перехода к конкретному опроснику нужно нажать на один из номеров в верхней части страницы
  • Список ключевых слов: страница со списком всех ключевых слов. При нажатии на букву вверху можно попасть к списку слов на эту букву. В нижней левой части есть ссылка "наверх", чтобы попасть обратно в шапку страницы. При нажатии на слово в новой вкладке открывается список текстов, где есть такое ключевое слово.
Галерея
В архиве есть галерея, где собраны фотографии, сделанные во время экспедиций.

  • Меню галереи. Вы можете выбрать фотографии по географическому признаку или по ключевых словам.

  • Страница фотографий. Вы можете увеличить фотографию, нажав на нее. Увеличенная фотография появится во всплювающем окне. Закрыть его можно, нажав на любую область вне фотографии. Под фотографиями располагаются теги, нажав на которые можно попасть на страницу фотографий по этому тегу.
Полнотекстовый поиск
В настоящее время работает полнотекстовый поиск.
  • Поиск по точной форме
  • Поиск по лемме
Для поиска наберите слово в одном из полей. Если вы хотите найти сочетание слов в предложении, то добавьте слово с помощью знака +. Для поиска нажмите Enter или кнопку Искать. В выдаче при нажатии на заголовок текста вы попадете на полный текст. При нажатии на стрелочки справа показываемый контекст расширяется.
